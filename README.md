# Text::Chart

A simple Text Chart for Raku


## Running an example

`raku-text-chart` should have been installed in your binary directory, you 
can run it directly this way:

```shell
raku-text-chart 3 2 1 4
```

Obtaining a result such as this one:

```text
    
   ▮
   ▮
▮  ▮
▮  ▮
▮  ▮
▮▮ ▮
▮▮ ▮
▮▮▮▮
▮▮▮▮

```
